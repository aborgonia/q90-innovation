# q90-innovation app

## Installation

Install docker and run on your local machine

[Download Docker](https://www.docker.com/)

Clone this repo then 
```
cd q90-innovation
```

#### Install with yarn

To install packages, run the following command:

```
docker exec -it app /bin/sh
yarn install
```
